# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-16
### Added
- Initial release of BloxLogicAI.
- AI Forecasting with Facebook Prophet for predicting Tea Exports and Market Demand.
- Anomaly Detection using Scikit-Learn Isolation Forest for tracking supply chain disruptions.
- Custom SHA-256 Blockchain implementation for secure traceability of tea batches.
- Responsive Streamlit web interface with distinct User and Admin portals.
- Comprehensive user dashboard, interactive charts, and dataset management tools.
